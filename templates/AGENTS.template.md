## Project Rules

- This is a data-led pSEO site, not a generic content app.
- Work in this order: data -> build/runtime -> UX -> SEO -> trust/compliance -> performance -> conversion -> docs.
- Do not let full mathematical rankings become the only default user view.
- Keep indexing layered and controlled.
- Keep route helpers and scaffold config as the single source of truth for starter identity.
- Keep the skill surface small. Prefer workflow docs and `docs/solutions/`.
