# Override Registry

Track every manual override separately from raw or transformed source data.

| Override id | Scope | Reason | Value changed | Date added | Reviewer | Recheck by | Status |

|  |  |  |  |  |  |  | active |

## Rules

- Do not mix overrides into source registry entries.
- Every override must state why automation was insufficient.
- Every override must have a recheck date.
- Retire stale overrides when upstream data catches up.
